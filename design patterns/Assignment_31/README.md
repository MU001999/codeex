# Run it

Just type `make` and enter, then type './bin/assignment_31' and enter, you will see the output.

## For example

```plain
> make
mkdir bin
g++ -std=c++17 Test.cpp -o bin/assignment_31
> ./bin/assignment_31
I'm a rat, I run away LOL
I'm a dog, I bark LOL
I'm a duck, I do nothing LOL
...
> make clean
rm -rf ./bin
```
